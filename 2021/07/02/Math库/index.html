三角函数

sin()方法

sin()方法用于返回指定弧度的正弦值

math.sin(x)

参数说明：

  • x:表示指定的数值
  • 返回值：返回x的正弦值
import math
print(math.sin(math.pi/6))
print(math.sin(-math.pi/6))
print(math.sin(0))
print(math.sin(math.pi/2))
'''
0.49999999999999994
-0.49999999999999994
0.0
1.0
'''

cos()方法

cos()方法用于返回指定弧度的正弦值

math.cos(x)

参数说明：

  • x:表示指定的数值
  • 返回值：返回x的余弦值
import math
print(math.cos(math.pi/3))
print(math.cos(-math.pi/3))
print(math.cos(0))
print(math.cos(math.pi))
print(math.cos(math.pi * 2))
'''
0.5000000000000001
0.5000000000000001
1.0
-1.0
1.0
'''

tan()方法

tan()方法用于返回指定弧度的正弦值

math.tan(x)

参数说明：

  • x:表示指定的数值
  • 返回值：返回x的正切值
import math
print(math.tan(1))
print(math.tan(-1))
print(math.tan(0))
print(math.tan(math.pi/4))
'''
1.5574077246549023
-1.5574077246549023
0.0
0.9999999999999999
'''

asin()方法

asin()方法用于获取指定值的反正弦值

math.asin(x)

参数说明：

  • x:表示-1到1之间的数值。如果x的值大于1会产生一个错误
  • 返回值：返回x的正弦值
import math
print(math.asin(-0.5))
print(math.asin(0))
print(math.asin(-1))
print(math.asin(1))
'''
-0.5235987755982989
0.0
-1.5707963267948966
1.5707963267948966
'''

math.acos(x)

  • x:表示-1到1之间的数值。如果x的值大于1会产生一个错误
  • 返回值：返回x的余弦值

atan()方法

atan()方法用于获取指定值的反正切值

math.atan(x)

参数说明：

  • x:表示-1到1之间的数值。
  • 返回值：返回x的正切值

atan2()方法

atan2()方法返回给定x，y坐标值得反正切值

math.atan2(y,x)

参数说明：

  • y，x:表示指定的数值
  • 返回值：返回给定x，y坐标值得反正切值
import math
print(math.atan2(-0.50,-0.50))
print(math.atan2(0.50,0.50))
print(math.atan2(5,5))
print(math.atan2(-10,10))
print(math.atan2(10,20))
'''
-2.356194490192345
0.7853981633974483
0.7853981633974483
-0.7853981633974483
0.4636476090008061
'''

幂函数与对数函数

pow()方法

pow()方法用于获取x的y次幂

math.pow(x,y)

参数说明：

  • x：表示底数
  • y：表示幂指数
  • 返回值：返回x的y次幂，即x**y
import math
print(math.pow(2,3))
print(math.pow(2,0))
print(math.pow(3,2))
'''
8.0
1.0
9.0
'''

log()方法

log()方法用于获取指定值的自然对数

math.log(x[,base])

参数说明：

  • x：指定的数值
  • base：表示底数，默认为e
  • 返回值：返回x的自然对数
import math

print(math.log(1))
print(math.log(math.e * math.e))
print(math.log(math.pi))
print(math.log(81,9))
'''
0.11332868530700327
0.0
2.0
1.1447298858494002
2.0
'''

数论与表示

ceil()方法

返回指定数值的上舍整数

math.ceil(x)

参数说明：

  • x：指定的数值

  • 返回值：返回一个大于或等于x的最小整数。如果x不是float，则委托给x.__ceil()__，它应该返回一个Integral值。

import math
print(math.ceil(-50.06))
print(math.ceil(-20.23))
print(math.ceil(math.pi))
'''
-50
-20
4
'''

floor()方法

返回指定数值的下舍整数

math.floor(x)

参数说明：

  • x：指定的数值

  • 返回值：返回一个大于或等于x的最小整数。如果x不是float，则委托给x.__ceil()__，它应该返回一个Integral值。

import math
print(math.floor(-50.06))
print(math.floor(-20.23))
print(math.floor(math.pi))
'''
-51
-21
3
'''

用于获取指定数值的绝对值

math.fabs’(x)

  • x：指定的数值

  • 返回值：返回x的绝对值

import math
print(math.fabs(-5))

fmod()方法

用于获取两个数值的余数，返回值类型为浮点数

math.fmod(x,y)

参数说明：

  • x：表示被除数

  • y：表示除数

  • 返回值：返回x/y的余数，返回值类型是浮点数

import math
print(math.fmod(7,3))
print(math.fmod(22,6))
print(math.fmod(4,2))
'''
1.0
4.0
0.0
'''

isnan()方法

判断是否为数值，不是数字返回True，否则返回False

math.isnan(x)

参数说明：

  • x：表示需要判断的内容
  • 返回值：如果x不是数字返回True，否则返回False
import math
print(math.isnan(100))
print(math.isnan(0.0))
print(math.isnan(float('NaN')))
'''
False
False
True
'''

isclose()方法

判断两个值是否接近或相等，如果两个值接近或相等返回True,否则返回False

math.isclose(a,b,rel_tol=1e-09,abs_tol=0.0)

参数说明：

  • a，b：需要判断的两个值
  • rel_tol：相对容差
  • abs_tol：最小绝对容差
  • 返回值：当两个值接近（指定容差）或相等返回True,否则返回False
import math
print(math.isclose(5,6))
print(math.isclose(5,6,rel_tol=1))
print(math.isclose(6,6))
print(math.isclose(3.14,3.21))
print(math.isclose(3.14,3.21,rel_tol=0.7))
'''
False
True
True
False
True
'''

modf()方法

获取指定数值的小数部分与整数部分，两部分数值符号与x相同，整数部分以浮点型表示

math.modf(x)

参数说明：

  • x：指定的数值

  • 返回值：返回x的小数部分与整数部分

import math
print(math.modf(10.18))
print(math.modf(4.7))
print(math.modf(2))
print(math.modf(math.pi))
'''
(0.17999999999999972, 10.0)
(0.7000000000000002, 4.0)
(0.0, 2.0)
(0.14159265358979312, 3.0)
'''
