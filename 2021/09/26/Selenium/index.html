WebDriverWait()

显示等待

WebDriverWait()有四个参数

  • driver：WebDriver实例对象
  • timeout：超时时长，即最长等待时长，单位为秒
  • poll_frequency：调用频率，即以多长时间的周期执行判断条件，单位为秒
  • ignored_exceptions：在执行过程中忽略异常，默认只忽略NoSuchElementException

WebDriverWait()提供了until()和util_not()两个方法

  • until(method,message=””)：该方法会在指定等待时间内，每隔一段时间调用一次method，直到条件为True，如果超时，就抛出异常。
  • until_not(method,message=””)：该方法会在指定等待时间内，每隔一段时间调用一次method，直到条件为False，如果超时，就抛出异常。

implicitly_wait()

隐式等待

唯一的参数为设定的等待的时间

当页面全部元素显示出来后，才会执行后续语句，若此时未到指定时间，则提前终止，继续执行后续语句；反之抛出异常。

高亮显示正在操作的元素

def HighLight(driver,element):
    #封装好的高亮显示页面元素的方法
    #使用JS对传入的页面元素对象边框颜色循环三次
    #边框颜色先蓝后红，中间各停留1s，产生闪烁效果
    for i in range(0,3):
        driver.execute_script("arguments[0].setAttribute('style',arguments[1]);"
                              ,element,"border:2px solid bule;")
        sleep(1)
        driver.execute_script("arguments[0].setAttribute('style',arguments[1]);"
                              ,element,"border:2px solid red;")
        sleep(1)
        #恢复页面元素
        driver.execute_script("arguments[0].setAttribute('style',arguments[1]);",element,"")

为页面元素捕获异常

import traceback
from selenium import webdriver

driver = webdriver.Chrome()
driver.implicitly_wait(10)
driver.maximize_window()
driver.get("https://cn.bing.com/")
try:
    #名为qqq的页面元素不存在，所以会产生异常
    input_search = driver.find_element_by_name('qqq').send_keys('电子科技大学')
    btn_go = driver.find_element_by_name('go').click()
except Exception as e:
    driver.save_screenshot('err.png') #截屏方法
    print(traceback.print_exc())
driver.quit()

框架元素的切换

有的网页会有一个FramSet（框架集），在里面的元素并不能被直接定位到

from selenium import webdriver

driver = webdriver.Chrome()
driver.maximize_window()
driver.get("http://127.0.0.1:1080/webtours")
driver.switch_to.frame(1)
driver.switch_to.frame(0)
driver.find_element_by_name('username').send_keys('uestc')
driver.find_element_by_name('password').send_keys('123456')
driver.switch_to.parent_frame()
driver.switch_to.frame(1)
driver.find_element_by_partial_link_text('sign').click()
