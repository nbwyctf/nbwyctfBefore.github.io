初始化

seed()方法

实现随机的相同数值
random.seed(a=None,version=2)

  • a：表示需要指定的种子，若未提供a或a=None，则使用系统时间作为种子。种子相同时可随机生成相同的数值
  • version：默认为2，如果参数a为整数将不计较参数version对应的值是否相同；如果参数a为字符型，需要参数version对应的值相同
import random

random.seed()
print('默认种子',random.random())
random.seed()
print('默认种子2',random.random())
random.seed(a=1)
print('整数种子',random.random())
random.seed(a=1)
print('整数种子2',random.random())
random.seed(a='1',version=1)
print('字符种子',random.random())
random.seed(a='1',version=1)
print('字符种子2',random.random())
'''
默认种子 0.9410040055189969
默认种子2 0.8292736431806521
整数种子 0.13436424411240122
整数种子2 0.13436424411240122
字符种子 0.06984368455225554
字符种子2 0.06984368455225554
'''

整数

randrange()方法

用于获取指定范围内的随机整数

randrange(stop)

randrange(start,stop[,step])

  • stop：在未指定start时表示0至stop之间的随机整数，不包含stop指定的值
  • start：表示随机范围的起始值。指定该参数后，stop参数将作为随机范围的结束值
  • step：表示步长
  • 返回值：返回指定范围的随机整数
import random
print(random.randrange(5))
print(random.randrange(1,6,2))#打印1到6之间的随机整数，其中不包含6，步长为2，即随机整数将在1和3之间出现
'''
1
5
'''

randint()

用于获取指定范围的随机整数

random.randint(a,b)

  • a：表示随机整数范围的起始值

  • b：表示随机整数范围的结束值，随机出现的整数包含b参数

  • 返回值：返回指定范围的随机整数

import random
print(random.randint(0,2))
#2

序列

choice()方法

从非空序列中返回一个随机元素

random.choice(seq)

  • seq：表示需要随机抽取的序列

  • 返回值： 从非空序列中返回一个随机元素

import random
num_list = [1,2,3,4,5,6]
str_list = ['a','s','d','m']
print(random.choice(num_list))
print(random.choice(str_list))
'''
3
a
'''

choices()方法

是choice()的升级版

random.choices(population,weights=None,*,cum_weights=None,k=1)

  • population：表示集群中随机抽取k个元素
  • weights：表示相对权重列表
  • cum_weights：表示累计权重，该参数与weights参数不能同时存在
  • 返回值：返回序列的随机元素
import random
print((random.choices(['apple','orange','banana'],k=2)))
#['orange', 'orange']

shuffle()方法

用于将指定序列的元素随机排列，只针对可变序列

random.shuffle(x[,random])

  • x：表示需要随机排列元素的序列
  • random：可选参数，返回[0.0,1.0]中的随机浮点，默认情况下该参数为random()方法
import random
num_list = [1,2,3,4,5,6]
random.shuffle(num_list)
print(num_list)
#[4, 6, 1, 2, 5, 3]

sample()方法

返回无重复随机数列表

random.sample(population,k)

  • population：表示需要抽取的可变序列或不可变序列
  • k：表示抽取元素的个数
  • 返回值：返回在指定序列中抽取指定数量且不重复元素形成的新序列
import random
num_list = [1,2,3,4,5]
str_tuple = ('6','7','8','9','0')
print(random.sample(num_list,3))
print(random.sample(str_tuple,3))
'''
[3, 2, 5]
['7', '9', '8']
'''

random.random()

import random
