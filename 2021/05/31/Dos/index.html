打开CMD的方式

  1. 开始+系统+命令提示符
  2. Win键 + R 输入cmd打开控制台
  3. 在任意的文件夹下面,按住shit键+鼠标右键点击,在此处打开命令行窗口
  4. 资源管理器的地址栏，选择所有地址，删除并输入cmd

盘符切换

  • 查看当前目录下的所有文件 dir

  • 切换目录 cd change directory

  • 返回上一级目录 cd ..

  • 清理屏幕 cls(clear screen)

  • 退出终端exit

  • 查看电脑的ipipconfig

  • 打开应用（exp: calc、mspaint、notepad）

#ping命令

exp ： ping www.baidu. com

#文件操作

  • 创建文件夹 ：md 目录名
  • 移除文件：rd 文件名
  • 创建文件：cd> 文件名
  • 删除文件：del 文件夹
