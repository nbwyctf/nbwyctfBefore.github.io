快捷打代码

  • 快速打public static void main(String[] args){}

    打psvm 回车 或者psvm Tap

  • 快速打System.out.println()

    打sout 回车

  • 快速new一个变量

    alt+shift+L

  • 快速for循环

    fori+Enter(回车)

  • 快速补全new 一个变量后的当行代码

    Alt+Enter(回车)

  • 代码补全

    Ctrl + Shift + Enter(回车)

  • 补全缺失的方法(生成构造器)

    Alt+insert

IDEL快捷键

  • Ctrl+F12: 快速浏览当前编辑的文件结构

  • Shift+Shift: 通过Search Everywhere窗口计算简单的算术表达式。

  • Alt+Insert: 为类中的字段生成getter和setter方法

  • Ctrl+两次空格: 速完成位于项目、库或 JDK 中任何位置的静态方法的方法调用\

  • Ctrl+J: 要查看带有后缀模板的建议列表

  • Ctrl+Alt+Shift+Insert: 要创建临时文件

  • Ctrl+Shift+空格 : 成分析整个表达式的预期类型，并帮助找到适用于当前上下文的方法和变量。 它在 return 关键字之后、赋值中、方法调用的参数列表中以及其他地方起作用。

  • Ctrl+Alt+B: 导航到抽象方法的实现

  • Ctrl+空格: 在声明变量时，代码完成可以为变量提供一个名称

  • Ctrl+Alt+Shift+I: 快速找到并运行检查

  • Ctrl+Alt+V: 提取变量重构将选定的表达式封装到变量中。它添加一个新的变量声明，并使用表达式作为初始化式

  • Ctrl+Alt+Shift+N : 快速打开编辑器中的任何类、字段或函数

  • Ctrl+Shift+J: 将两行合并为一行并删除不必要的空格

  • Ctrl+Alt+S: 从建议列表中选择一个动态模板时，该模板将扩展为在设置/首选项对话框,动态模板页面上定义的变体。

  • Ctrl+Shift+UP/DOWN: 整行代码向上下移动

  • Ctrl+Q: 快速查看插入符号处的类或方法的文档

  • Ctrl+H：显示继承的父类
